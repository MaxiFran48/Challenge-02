## Instrucciones de uso
1. Clonar el repositorio en su maquina
2. Abri la carpeta en alguna terminal
3. Desde la raiz de la carpeta, ejecutar el siguiente comando:
    ```
    ./gradlew clean build
    ```
4. El ejecutable final estara dentro de "build/libs/"
5. Para ejecutar el JAR final ejecute el siguiente comando:
    ```
    java -jar build/libs/Challenge-02-1.0-SNAPSHOT.jar
    ```
